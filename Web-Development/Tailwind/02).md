### What it does

* Injects **base styles** (also called *Preflight*)
* Normalizes browser styles (like a modern CSS reset)

### Includes

* Removes default margins
* Makes fonts consistent
* Sets `box-sizing: border-box`
* Normalizes headings, buttons, inputs, etc.

Example:

```css

  margin: 0;
  font-family: system-ui, sans-serif;
}
```
